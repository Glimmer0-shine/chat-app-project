## Project Composition
本プロジェクトは、フロントエンドと Supabase (Backend as a Service) を直接連携させることで、リアルタイムなチャット機能を実現しています。

frontend/: React + Supabase SDK

backend/: Python (Flask)

注：現在はフロントエンドとSupabaseで完結していますが、今後「AIによる会話要約」や「画像解析」などの高度な機能を実装するための拡張用基盤として用意しています。
